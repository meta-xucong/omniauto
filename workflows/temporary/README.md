# Temporary Workflows

This directory stores one-off or occasional workflows that are useful to keep, but are not yet promoted as reusable workflow assets.

## What Belongs Here

1. ad hoc task scripts
2. exploratory task variants
3. narrow utility workflows that are not stable enough for `examples/`, `verification/`, or `skills/`

## Promotion Rule

Temporary workflows may automatically contribute to:

1. `knowledge/tasks/`
2. `knowledge/patterns/`
3. `knowledge/lessons/`
4. `knowledge/capabilities/`

They do not automatically become:

1. formal skills
2. platform infrastructure

Those hard landings require explicit user approval.
