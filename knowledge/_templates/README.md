# Knowledge Templates

These templates keep task closeout consistent.

Use them when a task graduates from "we ran something once" to "we want future humans and AI to understand and reuse it."

## Templates

- `task_record_template.md`
  - For one completed initiative or task family.
- `pattern_template.md`
  - For reusable workflows or operating patterns.
- `lesson_template.md`
  - For durable pitfalls and maintenance notes.
- `capability_template.md`
  - For a stable capability entry backed by evidence.
