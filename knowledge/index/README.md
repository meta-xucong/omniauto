# Knowledge Indexes

This directory is optimized for fast lookup.

## Files

- `task_catalog.md`
  - Fast list of completed initiatives and where their assets live.
- `capability_matrix.md`
  - Short capability summary with maturity and evidence.
- `pattern_index.md`
  - Summary view of reusable and emerging patterns.
- `lesson_index.md`
  - Summary view of durable lessons by domain.
- `knowledge_registry.json`
  - Machine-readable registry for scripts or AI tooling.
- `proposal_queue.md`
  - Current human-facing queue of auto-generated hard-landing proposals.
- `ai_candidate_queue.md`
  - Review-only AI candidate queue, separate from the formal knowledge and proposal layers.
- `promotion_queue.md`
  - Compatibility pointer for older links.

If you only have one minute to orient yourself, start with `task_catalog.md` and `capability_matrix.md`.
