# Capabilities

This directory stores the evolving capability view of the project.

## Subdirectories

- `observed/`
  - capabilities supported by current task, script, output, or test evidence
- `candidate/`
  - capabilities that may deserve future hard landing

## Important Boundary

Observed or candidate capabilities are still part of the repository's soft-upgrade layer.

They do not automatically become:

1. formal skills
2. platform infrastructure
3. platform-level stability guarantees

## Current Default

The current human-readable capability map lives under `observed/`.
