# Durable Lessons

This directory stores long-lived lessons that should stay visible during maintenance and onboarding.

Use `lessons/` for:

1. Repeated failure modes
2. Environment and tooling gotchas
3. Boundary decisions that future work should not rediscover the hard way

Keep each lesson short and operational.

Store lessons under a domain bucket instead of directly under `lessons/`:

- `browser/`
- `desktop/`
- `marketplaces/`
- `platform/`
- `general/`
