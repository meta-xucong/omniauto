# 江苏车金二手车账号微信真人手测表 2026-05-04

## 使用方式

测试账号：二手车 customer 账号。  
发送账号：微信账号“许聪”。  
建议方式：每条单独发送，等系统回复后再发下一条；“上下文追问组”需要连续发送，不要中途换话题。

## 判定标准

- 正常覆盖：能结合商品库、正式规则和 RAG 经验自然回答，不像机械模板。
- 覆盖不到：不编造不存在的车源或政策，应说明当前没有明确信息，必要时转人工。
- 临近边界：可以解释常规流程，但不能承诺最低价、贷款包过、绝对无事故、赔偿、试驾安排、销售必联系、发票合同变通等。
- 模型预期：普通咨询优先 Flash；高风险、强承诺、金融/发票/合同/售后/事故水泡火烧等优先 Pro 或转人工。
