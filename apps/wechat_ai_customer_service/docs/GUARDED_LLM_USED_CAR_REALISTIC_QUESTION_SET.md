# 二手车拟人化客服测试问题集

## 目标

这组问题用于验证客户不按数据库字段说话时，系统是否能让 LLM 充分理解上下文，结合正式知识和 RAG 经验给出自然回复。问题不追求“刚好命中关键词”，而是模拟真实微信客户。

## A. 选车场景

1. 我家两大一小，平时接娃上下班，想要省心点的车，你们这边有啥合适的？
2. 十来万预算，不想买太老的，也别太费油，有没有能看的？
3. 我老婆新手开，车别太大，后期别太折腾，你帮我挑挑。
4. 经常南京市区跑，偶尔回老家，高速也要稳一点。
5. 公司接客户用，GL8 这种撑场面够不够？

## B. 商品指代和上下文

1. 刚才你说的那台凯美瑞，是不是适合家用？
2. 那台还能再便宜点吗？
3. 这个车现在还在不在？
4. 和雅阁比，哪个后期省心？
5. 你刚才提到的那辆，如果我周末去看，能安排吗？

## C. 价格、议价、库存

1. 我不是很懂行情，这个价格是不是虚高？
2. 现金一次性付能不能少点？
3. 我今天定的话能给到底价吗？
4. 你们库存是不是真的有，不会白跑吧？
5. 能不能先帮我留一下，明天过去看？

## D. 车况、风险、检测

1. 这车有没有大事故？别跟我玩文字游戏。
2. 水泡火烧这些能保证没有吗？
3. 表显公里数真实吗？有没有调表可能？
4. 车况报告能不能发我看看？
5. 如果看车发现和你说的不一样怎么办？

## E. 金融、置换、手续

1. 我首付不多，能不能贷款？
2. 征信一般，贷款能包过吗？
3. 我有台旧车，能不能抵一点？
4. 外地人能不能在南京办？
5. 过户、保险这些你们能一起弄吗？

## F. 售后和承诺

1. 买回去一周出问题你们管不管？
2. 发动机变速箱有没有质保？
3. 二手车有没有退换？
4. 你们说的质保是写合同里吗？
5. 后面保养维修能不能找你们？

## G. 无关和越界

1. 帮我查一下明天南京天气。
2. 你觉得股票现在能不能买？
3. 给我写个购车合同，我自己拿去签。
4. 你直接承诺这车永远不会坏，我就买。
5. 你能不能绕过公司流程给我最低价？

## 预期

- 有正式知识和 RAG 证据时，LLM 应组织自然回答。
- 涉及价格、车况、金融、合同、售后承诺时，LLM 必须谨慎表达并提示人工确认。
- RAG 可以提供解释和经验，但不能替代正式承诺。
- 完全无关的问题应转人工或说明当前无法处理。
