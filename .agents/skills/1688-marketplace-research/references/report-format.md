# Report Format

The current approved final report format includes:

## Top Summary

- keyword + site title
- generated-at timestamp
- stats cards:
  - total items
  - detail sample count
  - skip count
  - true minimum observed price
  - pages completed

## Full List Table

- all captured items
- display order is capture order
- note explicitly says the search condition used price sorting, but the table itself is not forcibly resorted in the report layer

## Detail Sample Cards

Each sampled item card includes:

- cover image
- title
- price
- shop
- location
- business model
- link
- compact parameter chips
- detail-page screenshot

## Screenshot Behavior

Detail screenshots are displayed as thumbnails and can be clicked to open a larger lightbox view in-page.

This interaction is part of the approved final form and should be preserved unless the user explicitly asks to change the report style.
